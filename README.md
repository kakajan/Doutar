# Doutar

This repository was initialized from the local workspace `Doutar`.

How to push locally-created repo to GitHub (requires `gh` CLI authenticated):

```bash
gh repo create Doutar --public --source=. --remote=origin --push
```

Or create an empty repo on GitHub and add remote manually:

```bash
git remote add origin https://github.com/kakajan/Doutar.git
git push -u origin main
```
